# 🛍️ Shop.co - WordPress E-commerce Homepage

A fully responsive and dynamic e-commerce homepage built using **WordPress (Custom Theme)** and **Advanced Custom Fields (ACF)**.

---

## 🚀 Features

* ✅ Custom WordPress Theme (No page builders)
* ✅ Fully Responsive Design (Mobile + Desktop)
* ✅ Dynamic Content using ACF
* ✅ Clean UI based on Figma Design
* ✅ Modular structure (Header, Footer, Sections)
* ✅ Editable content via WordPress Admin

---

## 🎯 Sections Implemented

* 🔹 Announcement Bar
* 🔹 Header (Navigation + Search + Icons)
* 🔹 Hero Section (Dynamic text, image, stats)
* 🔹 Brand Showcase
* 🔹 New Arrivals
* 🔹 Top Selling Products
* 🔹 Browse by Dress Style
* 🔹 Customer Reviews
* 🔹 Newsletter Section
* 🔹 Footer (Fully Dynamic)

---

## ⚙️ Tech Stack

* HTML5
* CSS3
* WordPress
* PHP
* Advanced Custom Fields (ACF)

---

## 🧠 Dynamic Functionality

All major sections are controlled from the WordPress admin using ACF:

* Hero Content (heading, subheading, image, stats)
* Products (name, price, images)
* Customer Reviews
* Newsletter Content
* Footer Links & Content

## 🛠️ Setup Instructions

1. Clone the repository
2. Place the theme inside:

   ```
   wp-content/themes/
   ```
3. Activate the theme in WordPress
4. Install and activate **Advanced Custom Fields (ACF)**
5. Create required fields (as per project)
6. Set Homepage as static front page
